## Start the project

First, you have to have Node.js installed.

### Clone the app 
`git clone https://github.com/vanabful/Star-Wars-quiz.git`

### Navigate to hci
`cd Star-Wars-quiz`

### Run development server
`npm install`
`npm start`

Your page should be loaded at http://localhost:3000
